# Snips

A small, always-available Windows desktop app for storing and pasting reusable rich-text
snippets, with placeholder variables like `{{date}}`, `{{clipboard}}`, and `{{input:Name}}`.

Full design and rationale: [docs/SPEC.md](docs/SPEC.md).

## Status

Early scaffolding. Not yet usable.

## Building

Requires the .NET 10 SDK.

```
dotnet build
dotnet test
```
